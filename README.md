# Implementation of Internationalization(i18n) using ReactJS

## Cloning GitHub Repo 
```
git clone git@github.com:adilshehzad786/Ruby-118n.git
```
## Command to Install Dependencies
```
yarn 
```
## To Run Yarn Server
```
yarn run 
```


## To build ReactJS
```
react-scripts build
```
## To Eject ReactJS
```
react-scripts eject
```
## To Start ReactJS
```
react-scripts start
```
## To Test ReactJS
```
react-scripts test
```
